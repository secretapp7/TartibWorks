# TartibWorks

TartibWorks is a lightweight, modern note-taking app inspired by Joplin. It allows you to:
- Write and save notes.
- Organize and manage your thoughts with ease.

## How to Run

### Frontend
1. Open `index.html` in a browser.

### Backend (Optional)
1. Install Node.js and run:
   ```bash
   node server.js
   ```
2. Access the API at `http://localhost:3000`.

## Features
- Simple and clean UI.
- Save notes dynamically.
